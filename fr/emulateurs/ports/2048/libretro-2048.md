## \*\*\*\*![](./rom-30098_640.png)**Roms**

### **Extensions supportées**

Les roms doivent avoir l'extension :

* .game

### **Emplacement**

Placez les roms comme ceci : 

> 📁recalbox
>
> > 📁share
> >
> > > 📁roms
> > >
> > > > 📁ports
> > > >
> > > > > 📁2048
> > > > >
> > > > > > 🗒 2048.game

Cette rom est fournie dans votre Recalbox.
